Astivetoolkit

Astive is a toolkit and server for Java that provides an extensible architecture for developing, maintaining, and deploying telephony apps over Asterisk.

Download this project as a .zip file Download this project as a tar.gz file

The Astive Toolkit Project

This is the official Git repository for the Astive Toolkit.

Requirements

  • Java 1.7+
  • Maven 3.x.x+
  • Asterisk 1.8+
  • Git (optional)

Getting Astive

A pre-compiled version of ATK can be found in the website. You can also build ATK from Github as follows:

git clone https://github.com/fonoster/astivetoolkit.git
cd astivetoolkit
./assembly

NOTE: The script will generate the Astive Server and the SDK. You can find the assemblies in folder 'dist'.
